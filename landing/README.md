# NeyborHuud Landing Page

A beautiful "Coming Soon" landing page for NeyborHuud - a community-driven neighborhood platform.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Building

```bash
npm run build
npm start
```

## Deployment

This project is deployed on Vercel. Push to GitHub and connect your repository to Vercel for automatic deployments.
